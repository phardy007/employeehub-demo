# EmployeeHub
### Workforce Management App — Prototype

A mobile-first workforce management application built for a traffic control services company as a cost-effective alternative to enterprise payroll platforms like ADP.

Designed, built, and deployed by [Pamela Hardy](https://pamelahardy.com) — UX/Product Designer & Developer.

---

## Overview

EmployeeHub gives field-based businesses a simple, owned alternative to expensive SaaS payroll tools. It was validated with real employees before development and built to replace a $3,000–$5,000/year ADP subscription with a one-time custom build.

---

## Features

**Employee View**
- Clock in / clock out
- Live GPS breadcrumb tracking during active shifts (10-minute intervals)
- Personal certification tracking with expiry alerts
- Announcements and direct messaging
- Works on any phone — no app download required

**Admin View**
- Live field status dashboard — see who's clocked in and where
- Edit or correct clock entries
- Certification matrix across all employees with expiry alerts
- Timesheet approvals
- CSV payroll report export
- Role-based access: Admin, Bookkeeper, Employee

---

## Tech Stack

- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Backend:** Supabase (PostgreSQL, Auth, Realtime)
- **Deployment:** GitHub Pages (prototype) / Vercel (production)
- **GPS:** Browser Geolocation API

---

## Status

This is a working prototype built to validate the concept with real end users. A production build would include hardened Supabase Auth, real employee accounts, live GPS tracking tied to user sessions, and deployment to a custom domain.

---

## Design

Clean, mobile-first UI built for field workers who need fast, intuitive access on any device. No frameworks — intentionally lean for performance and maintainability.
